# Zarinpal

Provides a payment gateway with Zarinpal in Moodle

## Setup

1. Register for Zarinpal
2. Install the plugin
3. Grab your Merchant ID
4. Configure the Zarinpal payment account in Moodle with the merchant id and your payment method
5. Add 'Enrolment on payment' to the Moodle courses that you want
6. Configure the enrolment method with the IRR currency

## Details

Zarinpal is an Iranian company that provides payment infrastructure for e-commerce businesses and facilitates the electronic transfer of funds. Zarinpal was founded in 2010 by Ali and Mostafa Amiri and is among the first companies to receive a “payment facilitator” (Paydar/Payment Service Provider) approval from Shaparak. Zarinpal enables instant online purchases, and today more than one million users make use of its services.
